# kode-runner
コードを実行するDiscord Bot

## ディレクトリ
* `discord-bot`
  * discord bot app
* `runtimes`
  * submodule [naotiki/kode-runner-runtimes](https://github.com/naotiki/kode-runner-runtimes)
* `server`
  * backend server
* `shared`
  * shared data model, services for RPC
